How To Create A Development Server Using Http Server

python, development, server, http, server

LogicalBranch / Tuesday, 15 September 2020
Artwork by Jordan Harrison, Updated on Tuesday, 15 September 2020

The Python standard library comes pre-installed with a module called http.server. Which you can use to create a virtual host for your project.

cd into your project directory and run the following command:

python3 -m http.server

Then open your web browser and visit http://localhost:8000.

If you'd like to change the port you can use the following command:

python3 -m http.server 7000

Now you'd visit http://localhost:7000 instead of http://localhost:8000.

Warning: don't use http.server for production because it doesn't implement necessary security checks.
