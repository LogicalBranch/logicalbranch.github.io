How To Setup A Development Server Using Flask

python, flask, development, server, routing, route, application, app

LogicalBranch / Monday, 31 August 2020
Artwork by Denny Bu, Updated on Monday, 31 August 2020

Start by installing python3 & pip3 using the apt package manager:

sudo apt install python3 python3-pip

Then install Flask:

pip3 install flask

Import the Flask class and the render_template method from the flask module:

from flask import Flask, render_template

Create a new instance of Flask:

app = Flask(__name__)

Create a new route that listens for requests to the root of your application:

@app.route("/")
def index():
  pass

Return a rendered index.html Jinja template:

@app.route("/")
def index():
  return render_template("./index.html")

Run the application in debug mode and listen for requests on the desired port:

port = 8000
app.run(debug = True, port = port)

If you've copied the code correctly you should end up with this:

from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def index():
  return render_template("./index.html")

port = 8000
app.run(debug = True, port = port)

Finally open a new browser and visit http://localhost:8000, you should see the rendered HTML template.
